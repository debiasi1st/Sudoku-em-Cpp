# Sudoku em C++

Projeto de um jogo de Sudoku desenvolvido em C++, executado em ambiente de terminal.

## Funcionalidades
- Tabuleiro 9x9 gerado dinamicamente a partir de uma matriz base
- Randomização do tabuleiro (cópia direta, transposta e inversões)
- Exibição formatada do tabuleiro no terminal
- Validação das jogadas do usuário
- Controle de tentativas
- Opção de saída durante o jogo

## Tecnologias utilizadas
- C++
- Matrizes
- Estruturas condicionais
- Laços de repetição
- Biblioteca `<ctime>` para geração de números aleatórios

## Objetivo
Projeto acadêmico com foco em lógica de programação, algoritmos e resolução de problemas.
